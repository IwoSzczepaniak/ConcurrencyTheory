Problem Pięciu Filozofów¶

Krótki opis rozwiązania¶

Zasadnicze rozwizanie - Java 17
Wizualizacja i analiza danych - Python 3.9.12

Zadanie 1 powodowao zakleszczenie, wic nie jest one przedmiotem analizy

Sposb rozwizania:

  • Widelce
    • "Sytuacj" widelcw zasymulowaem tworzc tablic bibliotecznych semaforw(java.util.concurrent.Semaphore) o nazwie fork. Widelec moe by tylko zajty albo wolny - wic naturalnym dla mnie byo zastosowanie tutaj semaforw.
  • Filozofowie
    • Do implementacji zachowa filozofw stworzyem klas wasnej implementacji (Philosopher lub DiningPhilosopher). Wszyscy filozofowie implemetuj intefejs Runnable, a kady kolejny dziedziczy po zad1.Philosopher, wyjtkiem jest tutaj zad6.DiningPhilosopher, ktry dziedziczy po zad5.DiningPhilosopher.

Gwn klas s odpowiednio: Main dla podpunktw 1-4 oraz Main_ dla podpunktw 5-6. W obu klasach wystarczy zmieni import klasy odpowiednio Philosopher(1-4) lub DiningPhilosopher(5-6) - i opcjonalnie zmieni nazw pliku wyjciowego, eby uruchomi program.

Domylnie nie s printowane zachowania filozofw, eby je jednak zobaczy, naley uy konstruktora, ktry na kocu przyjmuje dodatkowo warto logiczn - po wprowadzeniu true - wtki bd wypisywa to, co robi filozofowie.

Poniej zaprezentowane s wyniki dla 20, 30 i 50 filozofw oraz 2s, 5s i 10s wykonywania programw. Na osi poziomej przedstawiem numer zadania, a na osi pionowej redni czas oczekiwania wyliczony ze rednich czasw oczekiwania filozofw.

import matplotlib.pyplot as plt

def show_results(number_of_philosophers):
    zad_numbers = range(2, 7)  # Change the range to 2-6 for 5 boxplots
    file_contents = []
    
    for i in zad_numbers:
        with open(f"outputs/{number_of_philosophers}_zad{i}.txt", "r") as f:
            data = [float(line.strip()) for line in f]
            file_contents.append(data)

    plt.boxplot(file_contents, zad_numbers)
    plt.xlabel('Numer zadania')
    plt.ylabel('Czas (ns)')
    plt.title(f"Rozkady czasu dla {number_of_philosophers.split('_')[0]} filozofw\ndla {number_of_philosophers.split('_')[1]}s trwania programu")
    plt.show()
